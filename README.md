# Warchester

Warchester is a fantasy Web3 ecosystem combining streaming revenue, NFTs, and interactive games.

## Projects

• Strike & Shield – tournament based arcade battle game  
• Tiny Fighters – collectible character NFTs  
• Citizen Portal – player profiles and wallet access  
• Treasury Dashboard – streaming revenue and treasury tracking  

## Vision

Build a self-sustaining fantasy city where entertainment revenue powers tournaments, rewards, and community events.
